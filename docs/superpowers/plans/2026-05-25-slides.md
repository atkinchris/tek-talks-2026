# Hardware Hacking Talk - Slides Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build all HTML slides for the hardware hacking talk, plus a Puppeteer script to screenshot them to PNG for PowerPoint assembly.

**Architecture:** Each slide is a standalone HTML file in `slides/` that references a shared stylesheet at `slides/shared/style.css`. Photos go in `slides/assets/` (added manually by the speaker). A Node.js Puppeteer script at `screenshot.js` renders every slide to a numbered PNG in `output/`.

**Tech Stack:** HTML, CSS, Node.js, Puppeteer

## Task 3: Screenshot script

**Files:**
- Create: `screenshot.js`

- [ ] Create `screenshot.js`:

```js
const puppeteer = require('puppeteer')
const fs = require('fs')
const path = require('path')

async function main() {
  const slidesDir = path.join(__dirname, 'slides')
  const outputDir = path.join(__dirname, 'output')

  if (!fs.existsSync(outputDir)) fs.mkdirSync(outputDir)

  const files = fs.readdirSync(slidesDir)
    .filter(f => f.endsWith('.html'))
    .sort()

  const browser = await puppeteer.launch()
  const page = await browser.newPage()
  await page.setViewport({ width: 1920, height: 1080 })

  for (const file of files) {
    const filePath = path.join(slidesDir, file)
    const outPath = path.join(outputDir, file.replace('.html', '.png'))
    await page.goto('file://' + filePath)
    await page.screenshot({ path: outPath })
    console.log('Captured: ' + file)
  }

  await browser.close()
  console.log('Done. ' + files.length + ' slides exported to output/')
}

main().catch(console.error)
```
