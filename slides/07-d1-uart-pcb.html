UART

A serial debug port,
left behind by the manufacturer.

Universal Asynchronous Receiver-Transmitter

TX
RX
GND

Look for a row of 3–4 pads, often unlabelled, near the edge of the board.

TX
Transmit — data out from the device
RX
Receive — data in to the device
GND
Ground — shared reference voltage
07

The square pad is usually GND. Not sure which pin is which? A multimeter will show voltage fluctuations on active pads.
