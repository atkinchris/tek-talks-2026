Applying it

Connected. Got a shell.

U-Boot> printenv
bootargs=console=ttyS0,115200 root=/dev/mmcblk0p2 ...
# U-Boot bootloader shell. Let's dump the firmware.

U-Boot> ums 0 mmc 0
UMS: LUN 0, dev mmc 0, hwpart 0 ...

# On the host — the eMMC appears as a USB drive
$ sudo dd if=/dev/sdb of=firmware.bin bs=1M

ums turns U-Boot into a USB mass storage gadget — the host sees the eMMC as an external drive.
