Exception levels

Not all code runs equal.

EL0 — User land  •  Your app, games, shell
↓ can request up
EL1 — Kernel  •  Manages hardware, enforces access
↓ only accessible from here
Secure registers  •  Hardware encryption keys

A normal user-land program can't read the secure registers. We need to be the kernel. We need a kernel module — but the kernel won't load unsigned ones.
