What's inside a firmware image?

Bootloader
Kernel
Root filesystem

Bootloader

Runs first. Initialises the hardware, then loads the kernel. On this device, it's U-Boot.

Kernel

The operating system core. Manages memory, processes, and device drivers.

Root filesystem

Everything else — the applications, config files, and games. Usually the largest partition.

The firmware dump is all of these concatenated into one binary blob. We need to pull them apart.
